# myClaudeStuff

A personal collection of settings files and documentation for using [Claude Code](https://claude.com/claude-code).

## Contents

- **`settings.json`** — Claude Code permissions configuration (allow/deny lists for tools and bash commands).
- **`claude-code-permissions-guide.md`** — Reference guide for configuring Claude Code permissions.

## Usage

These files are for personal reference and to keep my Claude Code setup in sync across machines.
